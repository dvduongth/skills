# GDD Generator (PROJECT INTELLIGENCE MODE)

## Purpose
Auto-generate a Game Design Document skeleton from codebase and design patterns.

## Inputs
- Codebase snapshot, GDD references, architecture notes.

## Outputs
- GAME_DESIGN.md, concept docs, feature backlog.

## Workflow
1. Parse project structure. 2. Extract core gameplay loops. 3. Produce draft GDD.
