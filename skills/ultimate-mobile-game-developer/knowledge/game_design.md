# Game Design – Mobile Essentials

## Purpose
Capture core game design principles for mobile titles.

## Core Concepts
- Coro loop: loop description, progression curve, rewards cadence.
- State machine: match/move/wait transitions.
- Player progression: level, cosmetics, unlocks, monetization hooks.
- Data model: players, sessions, matches, events.
- Metrics: retention, ARPDAU, session length, crash rate.
